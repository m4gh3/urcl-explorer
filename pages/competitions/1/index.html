Each competition will be announced a day earlier and start at the time specified by the announcement post. The theme and specifications/restrictions will be released right when it starts. There will be no prize, but if genius solutions are presented to complex themes you may get the chance to get @Certified URCL Wizard rank.

To program your solution you may use any tool you want, from compilers, syntax highlighters, or emulators. The output must be a runnable urcl code.

The submitted program must be open source, and can be freely used. Taking credit for it is not allowed.

To submit your program send a share URL to the to be created thread along with a link to the source code (on github) and ask a mod to pin it for you.

If you also have the urcl online you can also use a link to that in your shared link.
example share link: https://bramotte.github.io/urcl-explorer/?srcurl=https://raw.githubusercontent.com/martandrMC/urcl-projects/main/minesweeper.urcl

This events theme is making a game that only outputs to the console.

limitations:

These limitations are optional but highly encouraged
  • use of %BUS is not allowed
  • use a maximum of 256 instructions
  • maximum total memory size of 256 words
  • a maximum of 10 general purpose registers
  • BITS == 8
  • RUN ROM

Ranking catagories

Ranking will be determined by voting for the following catagories at the end of the competition:
Authenticity Use of limitations
Gameplay
Visuals
Overall
